#Discord youtube video player

- !help for help
- !d    Stops and disconnects the bot from voice
- !fs   Skips the song currently playing
- !j    Joins a voice channel
- !p    Plays the requested YouTube URL or query
- !r    Resumes playback
- !s    Pauses playback
- !v    Changes the player's volume

#Requirements

- Python (Obviously)
- PyNaCl
- asyncio
- discord
- youtube_dl
